  • dpois(x, lambda)

Example What is the probability of making 2 to 4 sales in a week if the average sales rate is 3 per week?


# Using cumulative probability
ppois(q = 4, lambda = 3, lower.tail = TRUE) - 
ppois(q = 1, lambda = 3, lower.tail = TRUE)
#> [1] 0.616115

# Using exact probability
dpois(x = 2, lambda = 3) +
dpois(x = 3, lambda = 3) +
dpois(x = 4, lambda = 4)
#> [1] 0.6434504

Example

Suppose a baseball player has a p=.300 batting average. What is the probability of X<=150 hits in n=500 at bats? X=150? X>150?


# probability of x <= 150
ppois(q = 150, lambda = .300 * 500, lower.tail = TRUE)
#> [1] 0.52

# probability of x = 150
dpois(x = 150, lambda = .300 * 500)
#> [1] 0.033

# probability of x > 150
ppois(q = 150, lambda = .300 * 500, lower.tail = FALSE) 
#> [1] 0.48

library(ggplot2)

The Poisson distribution approximates the binomial distribution with Î»=np if n>=20 and p<=0.05.

Example What is the distribution of successes from a sample of n = 50 when the probability of success is p = .03?


library(ggplot2)

       subtitle = "Poisson approximates binomial when n >= 20 and p <= .05.",
