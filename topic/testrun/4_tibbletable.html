問題描述

資料型態為tibble,當檔案很大時,會自動截斷只列印出開頭部分,但是使用下面的程式碼遇到檔案很大的時候,仍然會試圖全部列出,不實際(久到以為是當掉)

  • 有問題的程式碼 
    knit_print.data.frame = function(x, ...) {
  res = paste(c(
    '{{< bstable table_class="w-50 m-auto" >}}',
    knitr::kable(x),
    '{{< /bstable >}}'
  ),
  collapse = '\n')
  asis_output(res)

}
# register the method
registerS3method("knit_print", "data.frame", knit_print.data.frame)
  • 暫時改成這樣 
    knit_print.data.frame = function(x, ...) {
  if(inherits(x,"tbl_df")){ #🏷
    NextMethod()
  }
  else {
  res = paste(c( 
    '{{< bstable table_class="w-50 m-auto" >}}',
    knitr::kable(x),
    '{{< /bstable >}}'
  ),    
  collapse = '\n') 
  asis_output(res)
  }
}
# register the method
registerS3method("knit_print", "data.frame", knit_print.data.frame)
