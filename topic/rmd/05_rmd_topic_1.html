只編譯一個RMD文件

使用套件中的內部函數,為:::,例如 blogdown:::build_rmds()。 所以如果要編譯單一的RMD文件,可以 blogdown:::build_rmds('content/foo.Rmd') 產生 content/foo.html 以後,如果我們已經啟動 blogdown::serve_site(),則會自動更新網站，否則可以手動 blogdown::hugo_build()。

😄兩個RMD變成一個


Rmd_bind <- 
    function(dir = ".",
    book_header = readLines(textConnection("---\ntitle: 'Title'\n---")))
{
    old <- setwd(dir)
    if(length(grep("book.Rmd", list.files())) > 0){
    warning("book.Rmd already exists")
    }
    write(book_header, file = "book.Rmd", )
    cfiles <- list.files(pattern = "*.Rmd", )
    ttext <- NULL
    for(i in 1:length(cfiles)){
    text <- readLines(cfiles[i])
    hspan <- grep("---", text)
    text <- text[-c(hspan[1]:hspan[2])]
    write(text, sep = "\n", file = "book.Rmd", append = T)
    }
    render("book.Rmd", output_format = "pdf_document")
    setwd(old)
    }
