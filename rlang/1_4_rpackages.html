搜尋套件中的檔案

對函數system.file()的理解為,輸入是以套件為根路徑,然後相對路徑,傳回的是絕對路徑。 範例

system.file("help", "AnIndex", package = "splines") # 🏷
#> [1] "C:/PROGRA~1/MICROS~4/RCLIEN~1/R_SERVER/library/splines/help/AnIndex"  #🏷

大致上的搜索步驟

  1. 列出套件stats所在的子目錄

    system.file(package = "stats")

    #> [1] "C:/PROGRA~1/R/R-41~1.1/library/stats"

  2. 知道了子目錄,可以列出其中的檔案

    dir(system.file(package = "stats"))

    #>  [1] "COPYRIGHTS.modreg" "demo"              "DESCRIPTION"      
#>  [4] "help"              "html"              "INDEX"            
#>  [7] "libs"              "Meta"              "NAMESPACE"        
#> [10] "R"                 "SOURCES.ts"        "tests"

  3. 上面知道有子目錄demo,可以再進一步看其中的檔案:

    dir(system.file("demo", package = "stats"))

    #> [1] "glm.vr.R" "lm.glm.R" "nlm.R"    "smooth.R"

library(help = "base") # 🏷

範例 detach_package()

討論內建函數detach_package()
使用方法：

detach_package(vegan)
detach_package("vegan", TRUE)#🏷
看一下detach_package()這個函數是怎麼寫的
detach_package <- function(pkg, character.only = FALSE)
{
  if(!character.only) #如果沒有指定參數是字串
  {
    pkg <- deparse(substitute(pkg)) # 轉字串
  }
  search_item <- paste("package", pkg, sep = ":")
  while(search_item %in% search())
  {
    detach(search_item, unload = TRUE, character.only = TRUE)
  }
}
