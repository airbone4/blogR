條件指令

if

一般來講，通用寫法如下：

if(判斷式){
 成功
}else{
 失敗
}

多行寫法

if(4 > 3){
  TRUE
}else{
  FALSE
}

另外，如果區塊內只有一行

if(4 > 3) TRUE else FALSE

ifelse的寫法

ifelse('判斷式','成功','失敗')
例如， ifelse(4 > 3, T, F)

switch的寫法

switch('判斷式', 
       '判斷值1'={}, 
       '判斷值2'={},
       ...,
       "其他判斷值"={}
       )

  • 直接指定哪一個

    switch(2,"red","green","blue")
#> [1] "green"
switch(1,"red","green","blue") #🏷
#> [1] "red"
y<-switch(1,"red","green","blue") #🏷
y
#> [1] "red"

  • 上面傳回字串,這裡傳回數字

    switch(2, 
      1, 
      2, #因為判斷式為2，因此，執行動作2
      3) #🏷
#> [1] 2

  • 也可以利用字串

    x="a"
switch(x, "a" = 1, "b" = 2, "c" = "why")
#> [1] 1
x="b"
switch(x, "a" = 1, "b" = 2, "c" = "why")
#> [1] 2
x="c"
switch(x, "a" = 1, "b" = 2, "c" = "why")
#> [1] "why"

  • 再一個例子

    x="c"
switch(x, "a" = 1, "b" = 2, "c" = {"why"}) #🏷
#> [1] "why"
x="c"
switch(x, "a" = 1, "b" = 2, "c" = {x<-3;x+1}) #🏷
#> [1] 4
x #🏷
#> [1] 3

這個switch 並不是一個語法，而是一個函數，原理大概是 第1個參數以後的東西，會被變成是一個named list。 例如上面的例子中

alist <-list(a=1,b=2,c="why")

然後利用判斷式 查找要返回對應的值。例如


switch("a", 
      a = 1,         
      b = 2,       
      c = "why")
#> [1] 1
switch("b", 
      a = 1,         
      b = 2,       
      c = "why")
#> [1] 2

迴圈指令

R 語言的迴圈，可以搭配 break和next，前者跳出迴圈，後者繼續下一個迴圈。

for-loop

計算 1+2+3+4. 的值是多少？

rst <- 0
for(i in c(1:4)){ 
  rst <- rst + i
}

while-loop

i <- 1
result <- 0

while(i < 5){  
  rst <- rst + i
  i <- i + 1
}

repeat-loop

i <- 1
rst <- 0

repeat{  
  
  if(i > 4) break 
  rst <- rst + i
  i <- i + 1
}
