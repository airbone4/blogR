基本操作

操作通常和資料結構相關,可以參考 資料結構

在R語言中,型態不須經過宣告(declared)。 一個變數的型態經由assignment的過程決定,即<-右邊的R-Objects。也就是在指派變數值的時候,同時決定了型態。

基本的 R-object有 −

  • Vectors
  • Lists
  • Matrices
  • Arrays
  • Factors
  • Data Frames

最簡單的是vector物件,atomic vector 有6種data types(有時也叫做 6 個classes)

範例：不同的運算子，在個別元素和整個向量的運算結果不一樣：

a = c(TRUE, FALSE)
b = c(FALSE, F)
a | b
#> [1]  TRUE FALSE # 🏷👏全部元素
a || b
#> [1] TRUE # 🏷👏只有第一個元素
a & b
#> [1] FALSE FALSE # 🏷👏全部元素
a && b
#> [1] FALSE # 🏷👏只有第一個元素

xor(a, b)
#> [1]  TRUE FALSE

a||b, a&&b 只比對a,b中的第一個元素。因此長度只有1個。✔如果是流程控制,大體上用的這兩種。

& 和 | 比對元素,傳回長度為大小最長的那個。

0為FALSE,而1為TRUE。在R中,判斷一個變數是否TRUE,要檢查好幾個地方,因此有提供內定的檢查函數✔isTRUE()

x <- c(TRUE,FALSE,0,6)
y <- c(FALSE,TRUE,FALSE,TRUE)
!x
x&y
x&&y
x|y
x||y #<span data-nopre class="codenote nopre">🏷<span data-nopre  class="tooltiptext" >👏常用於邏輯判斷式</span></span>

> x <- c(TRUE,FALSE,0,6)
#> [1] 1 0 0 6
> y <- c(FALSE,TRUE,FALSE,TRUE)
#> [1] FALSE  TRUE FALSE  TRUE
> !x
#> [1] FALSE  TRUE  TRUE FALSE #👺👺<span data-nopre class="codenote nopre">🏷<span data-nopre  class="tooltiptext" >👺和C家族的非0為真不一樣🤷‍♀️</span></span>

#> [1] FALSE FALSE FALSE  TRUE

#> [1] FALSE

x <- c(TRUE,FALSE,0,6,0) # 個數有５個
y <- c(T,TRUE,FALSE,TRUE)　#　個數有４個

解釋上面的結果

  1. 長度為甚麼是５
  2. 最後一個為甚麼是Ｔ
  3. 如果x <- c(TRUE,FALSE,0,6,3) 然後問 x & y

邏輯函數

is.family 函數，可以用來判定變數是什麼型態，傳回真假值，例如

a = c(1, 2, 3)
is.numeric(a)
#> [1] TRUE
is.factor(a)
#> [1] FALSE
is.character(a)
#> [1] FALSE

函數which()哪些元素不一致

A<-c(T,F,T,F)
C<-c(F,F,F,T)
which(A)
#> [1] 1 3
which(A != C)
#> [1] 1 3 4

函數all() 是不是向量內所有元素都是Ｔ

all(A==B)
#> Error in eval(expr, envir, enclos): 找不到物件 'B'
all(A==C)
#> [1] FALSE
all(A==A)
#> [1] TRUE
which(),all()裡面的參數其實都是邏輯向量。

註解：但是R語言的向量運算，涉及自動填充循環，因此注意下面的計算結果

D <- c("A","B","A","B")
E <- c("A","B")
all(D==E)
#> [1] TRUE
all(length(D)==length(E)) && all(D==E)
#> [1] FALSE

如何存取vector中的元素

元素索引可以利用 logical, integer or character vector.

如果利用整數索引,則從1開始.但是,如果索引值給的是負數(例如-2),則除了2號元素以外,都會被傳回。但是不能同時有正負。同時,浮點數會被truncated。

怎樣變動矩陣向量?

We can modify a vector using the assignment operator.

We can use the techniques discussed above to access specific elements and modify them.

If we want to truncate the elements, we can use reassignments.

> x
#> [1] -3 -2 -1  0  1  2
> x[2] <- 0; x        # 只改變第二個元素
#> [1] -3  0 -1  0  1  2
> x[x<0] <- 5; x   # 將小於0的值,都改成5
#> [1] 5 0 5 0 1 2
> x <- x[1:4]; x      # 只要1到4
#> [1] 5 0 5 0

How to delete a Vector?

We can delete a vector by simply assigning a NULL to it.

> x
#> [1] -3 -2 -1  0  1  2
> x <- NULL
> x
#> NULL
> x[4]
#> NULL

structure & attributes

函數structure(data, ...)

使用方法

先看一下範例

code="這是範例"
x=structure(list(src = code), ddd = 'source')     
x
#> $src
#> [1] "這是範例"

#> attr(,"ddd")  🏷這是第二個參數添進來的
#> [1] "source"

再看另一個範例

data="這是範例" 🏷只是字串
x=structure(data, ddd = 'source') 🏷添加屬性ddd
x 🏷看內容
#> [1] "這是範例"
#> attr(,"ddd")
#> [1] "source"
attributes(data) 🏷看看屬性
#> NULL 🏷data仍然是data,沒有新增的屬性
attributes(x) 🏷傳回來的不僅有原來的data,還多了一個屬性
#> $ddd
#> [1] "source"

白話文就是 假設data是一個物件,則structure(data,xxx="dd"),會根據這個物件再添加一個屬性xxx,然後傳回這個物件。

因此要注意的是class這個attribute。在Ｒ中，class有另一個意思，用來代表物件類別（例如要自訂類別，可以利用這個class)

data="string"
class(data)
#> character 🏷這是字串
x=structure(data,class="risk") #🏷👏現在物件x的類別不再是字串，而是類別risk
class(x)
#> risk  🏷此時的類別被變更成risk
