  • Vectors

範例：不同的運算子，在個別元素和整個向量的運算結果不一樣：

a = c(TRUE, FALSE)
b = c(FALSE, FALSE)
a | b
#> [1]  TRUE FALSE
a || b
#> [1] TRUE
xor(a, b)
#> [1]  TRUE FALSE

a||b 只比對a,b中的第一個元素。因此只有個長度。

& 和 | 比對元素,傳回長度為大小最長的那個。

0為FALSE,而其他數字為TRUE。

x <- c(TRUE,FALSE,0,6)
y <- c(FALSE,TRUE,FALSE,TRUE)
!x
x&y
x&&y
x|y
x||y

#> [1] 1 0 0 6
> y <- c(FALSE,TRUE,FALSE,TRUE)
#> [1] FALSE  TRUE FALSE  TRUE

#> [1] FALSE  TRUE  TRUE FALSE
> x&y
#> [1] FALSE FALSE FALSE  TRUE
> x&&y
#> [1] FALSE
> x|y
#> [1]  TRUE  TRUE FALSE  TRUE
> x||y
#> [1] TRUE
  • <-, <<-, = 向左指派值。
  • ->, ->> 向右指派值。

大部分情況下<- 和 = 在相同的環境下等價。

邏輯函數

函數identical(a,b)
a,b 是否相等,比較的是除了a,b的內容以外,💡還有a b 的型態。例如，identical(1,as.integer(1)) 結果是F,因為 前者是double,後者是整數。

A <- c("A", "B", "C", "D")
B <- A
C <- c("A", "C", "C", "E")

identical(A,C)
#> [1] FALSE

identical 允許只比較有效位數。

函數which()哪些元素不一致

which(A != C)
#> [1] 2 4

函數all()

all(A==B)
#> [1] TRUE
all(A==C)
#> [1] FALSE
which(),all()裡面的參數其實都是邏輯向量。

註解：但是R語言的向量運算，涉及自動填充循環，因此注意下面的計算結果

D <- c("A","B","A","B")
E <- c("A","B")
all(D==E)
#> [1] TRUE
all(length(D)==length(E)) && all(D==E)
#> [1] FALSE

練習：
將上面可能造成誤判的情況，寫成函數先判斷是否長度相等，然後判斷元素相等，可以和identical()比較。

a <- character(0)
identical(a, character(0)) # returns TRUE

identical(a, "") # returns FALSE
identical(a, numeric(0)) # returns also FALSE

元素索引可以利用 logical, integer or character vector.

如果利用整數索引,則從1開始.但是,如果索引值給的是負數(例如-2),則除了2號元素以外,都會被傳回。但是不能同時有正負。同時,浮點數會被truncated。

> x
[1]  0  2  4  6  8 10

> x[c(2, -4)]    # 不能混合正負
Error in x[c(2, -4)] : only 0's may be mixed with negative subscripts

[1] 2 4

How to modify a vector in R?

We can modify a vector using the assignment operator.

We can use the techniques discussed above to access specific elements and modify them.

If we want to truncate the elements, we can use reassignments.

> x
#> [1] -3 -2 -1  0  1  2
> x[2] <- 0; x        # modify 2nd element
#> [1] -3  0 -1  0  1  2
> x[x<0] <- 5; x   # modify elements less than 0
#> [1] 5 0 5 0 1 2
> x <- x[1:4]; x      # truncate x to first 4 elements
#> [1] 5 0 5 0

How to delete a Vector?

We can delete a vector by simply assigning a NULL to it.

> x
#> [1] -3 -2 -1  0  1  2
> x <- NULL
> x
#> NULL
> x[4]
#> NULL

data="這是範例" 🏷只是字串

x 🏷看內容 
#> [1] "這是範例"
#> attr(,"ddd")
#> [1] "source"

#> $ddd
#> [1] "source"

白話文就是 假設data是一個物件,則structure(data,xxx="dd"),會根據這個物件再添加一個屬性xxx,然後傳回這個物件。

因此要注意的是

data="string"
class(data)
#> character 🏷這是字串 
x=structure(data,class="xx")
class(x)
#> xx  🏷此時的類別被變更成xx
前 後
