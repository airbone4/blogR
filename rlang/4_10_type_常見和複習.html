相關問題

想要知道有甚麼資料庫可以用? 例如， 想要知道資料庫裏面有沒有'mtcars'

  1. 開始測試過程

    xx<-data() #🏷👏利用指令data()可以列出內建資料：

  2. xx 是甚麼?直接打入xx並沒有像其他變數一樣，在console顯示內容,而是出現另一個視窗：

    xx

  3. 想要把所有的資料集名稱列表,而步驟2無法在程式碼中表示，因此研究一下怎樣拿取上述x的內容:

    str(xx)
#> List of 4
#>  $ title  : chr "Data sets"
#>  $ header : NULL
#>  $ results: chr [1:104, 1:4] "datasets" "datasets" "datasets" "datasets" ...
#>   ..- attr(*, "dimnames")=List of 2
#>   .. ..$ : NULL
#>   .. ..$ : chr [1:4] "Package" "LibPath" "Item" "Title"
#>  $ footer : chr "Use 'data(package = .packages(all.available = TRUE))'\nto list the data sets in all *available* packages."
#>  - attr(*, "class")= chr "packageIQR"
is.data.frame(xx)
#> [1] FALSE
is.list(xx)
#> [1] TRUE
is.table(xx)
#> [1] FALSE
class(xx)
#> [1] "packageIQR"

  4. class()函數說明xx是一個packageIQR物件。str()可以看出是一個長度為4的list，其中results標籤，是一個字串矩陣，大小為104 x 4。其中4個欄位的名稱為："Package" "LibPath" "Item" "Title"（如下）：

    head(xx$results)
#>      Package    LibPath                              Item                    
#> [1,] "datasets" "C:/Program Files/R/R-4.1.1/library" "AirPassengers"         
#> [2,] "datasets" "C:/Program Files/R/R-4.1.1/library" "BJsales"               
#> [3,] "datasets" "C:/Program Files/R/R-4.1.1/library" "BJsales.lead (BJsales)"
#> [4,] "datasets" "C:/Program Files/R/R-4.1.1/library" "BOD"                   
#> [5,] "datasets" "C:/Program Files/R/R-4.1.1/library" "CO2"                   
#> [6,] "datasets" "C:/Program Files/R/R-4.1.1/library" "ChickWeight"           
#>      Title                                           
#> [1,] "Monthly Airline Passenger Numbers 1949-1960"   
#> [2,] "Sales Data with Leading Indicator"             
#> [3,] "Sales Data with Leading Indicator"             
#> [4,] "Biochemical Oxygen Demand"                     
#> [5,] "Carbon Dioxide Uptake in Grass Plants"         
#> [6,] "Weight versus age of chicks on different diets"

    如果覺得混亂,可以利用

    View(xx$results)

  5. 因為是矩陣,所以不能以「$」得到Item的內容?

    xx$results$Item
#> Error in xx$results$Item: $ operator is invalid for atomic vectors

    矩陣拿一欄用中括號

    head(xx$results[,'item'])
#> Error in xx$results[, "item"]: 下標超出邊界

  6. 為甚麼錯誤?
    因為大小寫,所以正確寫法為：

    head(xx$results[,'Item'])
#> [1] "AirPassengers"          "BJsales"                "BJsales.lead (BJsales)"
#> [4] "BOD"                    "CO2"                    "ChickWeight"

  7. 利用grep()可以拿到mtcars的位置：

    idx<-grep('mtcars',xx$results[,'Item'])
idx
#> [1] 72

  8. 確認

    xx$results[idx,'Item']
#>     Item 
#> "mtcars"

  9. 一行寫完

    xx$results[grep('mtcars',xx$results[,'Item'])]
#> [1] "datasets"

詞彙澄清

觀念

x0 <- list(1, c(TRUE, FALSE), c("a", "b", "c"))
x0

正確念法：
正確念法 指派一個list給變數x0，裡面有三個元素，第一個元素是純量1，第二個元素是邏輯向量，長度為2，內容分別是TRUE, FALSE，第三個元素是字元向量，長度為3，內容分別是"a", "b", "c"

names(l1) <- c("A","B","C")

錯誤念法
l1的名稱等於❌集合("A","B","C")

正確念法：
將l1 這個list 的前3個元素名稱，分別指定為 A,B,C

觀念

l1$x <- NULL

正確念法：
刪除元素名為x的元素。

常用帶名稱的list

# l0 <- list(1, c(TRUE, FALSE), c("a", "b", "c")) # 🏷👏比較用

l1 <- list(x = 1, y = c(TRUE, FALSE), z = c("a", "b", "c"))
l1$x
l1$y
l1$z
l1$m

念法 建立list變數l1,名稱分別是x,y,z,內容分別是.......
l1$y 為邏輯向量
l1$z 為字元向量
🏷: l1$x === l1[[1]]
