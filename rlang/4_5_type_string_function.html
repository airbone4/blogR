函數 paste

最簡單的範例

paste0("x","y","z")
#> [1] "xyz"

範例

s1 = c('hello','Copy') 
s2 = c('john','pen') 
print(paste(s1,s2,sep=" ",collapse="-"))

#> "hello john-Copy pen"
s1 s2
hello john
copy pen

範例解說:

  1. paste0(c(1,2),c(3,4),collapse='') 會有甚麼答案?
  2. 怎樣得到"1234"

結果為1324 (不是 1234)

  1. 先兩個參數之間的相連,結果是("13","24")，長度為2的向量。
  2. 然後如果還要成為一個向量,可以給定參數collaose="",因此上例的執行結果為 "1324"。
  3. 如果沒有給參數,collapse="any"則,不會collapse。例如上例如果不給collapse,則執行結果是("13","24")

💡 paste0()是paste()的簡化版。paste0()預設分隔字元(separator)為""(無法更改),如果要用其他的分隔字元必須使用函數paste()。

rst=paste0(c(1,2)) 
rst # 🏷✔rst=c("1","2") 長度為2(因為只有一個參數,又沒有指定參數collapse)
#> [1] "1" "2"
length(rst)
#> [1] 2

rst=paste0(c(1,2),c(3,4),collapse="*")
rst
#> [1] "13*24"
length(rst)
#> [1] 1

rst=paste(c(1,2),c(3,4),collapse="*")
rst
#> [1] "1 3*2 4"
length(rst)
#> [1] 1

R提供的字串函數(ascii)

  • nchar: string length
  • paste: concatenate strings
  • substr: substring
  • toupper: convert entire string to uppercase
  • tolower: convert entire string to lowercase
  • chartr: character map replacement (like "tr")
  • strtrim：trunates string

nchar, substr, toupper, tolower will accept string vectors as arguments and return vector results.

strtrim accepts both a vector of strings and a vector of truncation positions.

  • \': 等同於 "'" .
  • \": 等同於 '"'.
  • \n: newline.
  • \r: carriage return.
  • \t: tab character.

Note: cat() and print() 處理逃逸字元的方式不一樣如果要在螢幕上解讀上面的字串,需要的是 cat().

print("a\nb")
#> [1] "a\nb"
cat("a\nb")
#> a
#> b

練習

collapse 的範例,問下面的程式的執行結果

paste(c('A', 'B', 'C'), 1:7, sep="-", collapse="/")

# [1] "A-1/B-2/C-3/A-4/B-5/C-6/A-7"

應用範例macro

  1. version 1

    x<- "${1} is good"
gsub("\\$\\{1\\}","dog",x)
#> [1] "dog is good"

  2. version 2 改成函數

    mstr<-function (src,mto)
{
  return (gsub("\\$\\{1\\}",mto,src))

}
x<- "${1} is good"
mstr(x,"dog")
#> [1] "dog is good"

函數 strsplit()

利用逗號分解,然後提取KEY,VALUE。

str= "cmd=bstable,class=\"d-block w-50\",style=\"display:none\""
strsplit(str,",") #🏷如果str 是字串陣列,那麼N個字串就會有N個分割,這裡只有一個分割,因此產生1個節點,這個節點是3元素的字串陣列。
#> [[1]]
#> [1] "cmd=bstable"            "class=\"d-block w-50\"" "style=\"display:none\""
unlist(strsplit(str,",")) # 🏷產生向量
#> [1] "cmd=bstable"            "class=\"d-block w-50\"" "style=\"display:none\""
as.list(unlist(strsplit(str,",")))
#> [[1]]
#> [1] "cmd=bstable"
#> 
#> [[2]]
#> [1] "class=\"d-block w-50\""
#> 
#> [[3]]
#> [1] "style=\"display:none\""

misc

函數 get() assign() 用來利用字串而不是變數來存取物件。而 as.name() 可以用來將字串轉為變數。 (可以配合 substitute() 建立 expression 然後eval() 。

例如 assign("a",get("b")) 相當於 a<-b 。

比較常見的方法是利用list()，例如

obj<-list(a=1,b=2)
obj[["a"]]<-otherobj[["b"]]

此時obj[["a"]] 是對元素a的參考。
